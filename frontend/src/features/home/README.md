# Home Feature 도메인

홈 화면에서 로또 번호를 관리하고, 추첨을 진행하고 통계를 확인하는 기능을 담담하는 도메인 폴더입니다.

## 컴포넌트 목록
- `LotteryGrid.tsx`: 메인 화면의 그리드로, 분석되어 올라온 임시 번호 세트 생성 결과물을 보여주며 선택한 회차에 대한 수동 당첨번호 입력/검사 기능을 제공합니다.
- `LotteryCard.tsx`: 각각의 로또 1건 (6개 번호)을 보여주는 UI 모듈입니다.
- `AlgorithmsInfo.tsx`, `WinningStats.tsx`: 홈 화면의 최하단에 통계 및 알고리즘 요소를 표시해줍니다. (UI 컴포넌트)

## API 의존성
- `GET /api/drawings/recommend`: 최신 회차(또는 대상 회차)의 생성된 임시번호 세트 목록 10개를 가져옵니다.
- `POST /api/drawings`: 그리드에서 생성된 10세트의 번호를 사용자가 최종 확정(구매 처리)할 때 호출합니다. 
- `GET /api/drawings/by-no`: 당첨 확인 시 특정 회차에 존재하는 모든 임시번호 전체를 가져옵니다.

참고: 기존 winners 전용 API(`/api/winners`) 의존은 제거되어, 당첨번호는 클라이언트에서 수동 입력하여 사용합니다.
