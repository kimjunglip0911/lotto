# 연속 출현 분석 (`run-streak`)

## 목적

선택 회차 기준으로 1~45번이 **본번호 6개**에 몇 회 연속 포함되었는지 표·요약으로 보여 준니다.

## 실행 방법

- 루트에서 `npm run dev` 후 브라우저에서 연속 출현 분석 메뉴로 이동합니다.
- 프론트만 단독 실행 시: `cd frontend` 후 `npm run dev` (포트는 프로젝트 설정 기준, 예: 3010).

## 환경 변수

- `NEXT_PUBLIC_API_URL`: 백엔드 베이스 URL. 비우면 상대 경로로 같은 출처의 `/api/analysis/run-streak/...` 를 호출합니다.

## `logic` 폴더 구조

- `fetch/`: API URL·당첨 데이터 로딩.
- `streak/`: 본번호 추출·연속 출현 수 계산·조회 한 번에 묶기(`stSearch`).
- `hintLine.ts`: 검색창 상단 안내 문구.

## 주의

- 당첨 이력이 DB에 없으면 회차 목록이 비어 안내만 표시됩니다.
- 오프라인 최종 사용자 환경을 고려해, 새 패키지 없이 기존 의존성만 사용합니다.
