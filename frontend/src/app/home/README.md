# Home Module

## 현재 회차 분석 번호(30세트) 표시 규칙

- 분석 번호 카드는 30세트를 한 번에 보여주지 않고, 10세트씩 3개 영역으로 나누어 표시한다.
- 각 영역 헤더 좌측에는 현재 세트 범위(예: `1~10세트`)가 표시된다.
- 각 영역 헤더 우측에는 `10세트 다운로드` 버튼이 제공된다.

## 10세트 다운로드 동작

- 버튼 클릭 시 해당 영역의 카드 UI가 화면 스타일 그대로 PNG 파일로 다운로드된다.
- 파일명은 세트 범위 기준으로 저장된다. 예: `1~10세트.png`, `11~20세트.png`
- 다운로드 이미지는 카드 배경/번호 공/기법 라벨 등 실제 카드 디자인을 유지한다.
- 버튼 상태는 짧게 `다운로드 완료` 또는 `다운로드 실패`로 변경되어 결과를 알려준다.
