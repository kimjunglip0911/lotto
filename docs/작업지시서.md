**[요구 사항]**
1. 누적 번호 분석 기법을 통해 당첨 번호로 나올 확률이 가장 높은 번호 4개를 채택하는게 고도화의 최종 목표.
2. 백테스팅을 통해 회차별 당첨번호와 비교하여 어떠한 형태일때(가장많이 나온번호, 가장 안나온번호, 평균 등) 당첨번호가 나오는지를 찾아야함.
3. 최소 4개의 번호에서 매회차 1개 이상의 번호가 나오게 해야함.
4. 분석을 통해서 누적 번호 분석 기법에 어떤 로직을 추가하면 확률일 높아질지도 아이디어를 찾아야함.
5. 회차별 분석할떄 항상 당첨회차 -1 이어야함. (100회 당첨번호를 분석할떄는 1~99회까지 누적 번호를 분석.)
6. 현재 설정되어 있는 기간(전체 누적 제외)보다 더 정확한 패턴이 나오는 기간도 찾아야함.
7. frontend\src\app\analysis\accumulated-numbers 폴더에 백테스팅 가능한 스크립트 작성.
8. frontend\src\app\analysis\accumulated-numbers\누적번호분석리포트.md 분석 이후 결과를 리포트에 작성.
9. 제대로된 분석 기법을 만들때 까지 누적번호분석리포트.md 참고하여 계속 고도화 예정.
