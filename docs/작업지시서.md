**[요구 사항]**
1. 카이제곱 페이지에 전체 회차(기준 회차 제외) 편차 값을 5% 단위의 구간으로 분류하는 표 생성.
2. 왼쪽에는 - 구간을 오른쪽에는 + 구간으로 분류.
3. 당첨 번호와 비교하여 당첨번호가 어느 구간에서 나왔는지 회차별 %를 계산.
4. 한회차에서 6개의 번호가 나오든 1개의 번호가 나오든 1개만 나온걸로 계산.
5. 전체 회차 대비 %를 계산하여 높은 번호가 위로 갈수 있도록 정렬.
