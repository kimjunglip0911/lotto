**[Task]**
코드 최적화를 위한 리팩토링 작업

**[리팩토링 대상]**
1. frontend\src\app\analysis\run-streak\components

**[참고 문서]**
1. docs\로직.md

**[요구 사항]**
1. .cursor\rules\refactoring-guide.mdc 에서 요구하는 사항 필히 적용.
2. 의미없는 코드 삭제.
3. 비효율적 코드 수정.
4. 사용하지 않는 로직이 있다면 삭제.
5. 사용하고 있는 로직이 로직.md 에 작성되어 있지 않거나 틀리다면 최신화.
6. 각 파일마다 해당 파일이 어떠한 기능을 가지고 있는 파일인지를 가장 윗단(임포트 아래쪽)에 주석을 작성.
7. 주석은 비개발자가 보아도 이해가 가능하도록 쉽고 핵심만으로 작성.
8. 리팩토링 대상 밖에서 연결된 부분이 있다면 해당 부분도 같이 수정.
9. 공통 컴퍼넌트로 분리할수 있는게 있다면 분리.
10. 파일이 5개가 넘어간다면 비슷한 기능으로 묶어서 폴더 생성하여 이동.
